## ðŸ“ Layout Dimensions

### Desktop (> 1024px)
- **Container:** max-w-7xl (1280px)
- **Grid:** 3 columns (2:1 ratio)
- **Gap:** 1.25rem between cards
- **Padding:** 1.5rem inside cards

### Tablet (768px - 1024px)
- **Container:** Full width with padding
- **Grid:** 2 columns or stacked
- **Gap:** 1rem between cards

### Mobile (< 768px)
- **Container:** Full width
- **Grid:** Single column
- **Gap:** 0.75rem between cards
- **Padding:** 1rem inside cards
